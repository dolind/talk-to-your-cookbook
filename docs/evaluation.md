# Retrieval & LLM Evaluation Plan

Evaluation focuses on the effectiveness of recipe similarity search and grounding.


Existing: 
A mock chatbot enables testing retrieval without a generative model.


## Dataset

A small, anonymized recipe sample will be used for public benchmarks.  

## Metrics

- **Recall@K**
- **MRR**
- **Latency**
- **Grounding score**
- **Hallucination reduction**
